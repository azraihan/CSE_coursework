### Running The Script

```bash
bash run-script.sh sampleio_abs/<input_file_name>
```

e.g.
```bash
bash run-script.sh sampleio_abs/input1.txt
```

The output will be found in the `output/` directory:
- `errorLog.txt`
- `lexerLog.txt`
- `parserLog.txt`

### Cleaning the outputs

```bash
bash clean-script.sh
```
